## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/BasiliskAI-dev/PythonProject
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
3. Установите виртуальное окружение
```
python -m venv venv
```
4. Начните запускать функции
